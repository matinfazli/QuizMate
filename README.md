# Quiz Mate
Quiz making & doing application using ReactJS and Firebase.
---
* Real-time data update using Firebase & ReactJS
* Fully responsive using Bootstrap (and React-Bootstrap)


## 1. Install

```bash
$ npm install
```

## 2. Run

#### Webpack build for development with BrowserSync
```
$ npm run dev
```

#### Deploy
```
$ npm start
```
